purl のパスが Wayback に残っているか調べる

このページは archive.org/wayback/available を使って、 入力したパスが purl.org と purl.archive.org のどちらで保存されているかを確認します。 直近のスナップショットだけを返す API なので、一覧ではなく「最近の保存状態」を見るための検索です。
